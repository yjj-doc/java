## Amber 
https://openjdk.org/projects/amber/

: 지역 변수 유형 추론, 레코드, 패턴 매칭, 스위치 표현식 등 자바 언어 생산성 향상을 위한 프로젝트  

- JEP 286 : 지역 변수 유형 추론
- JEP 301 : 향상된 열거형
- JEP 405 : 레코드 패턴 
- JEP 409 : Sealed 클래스 
- JEP 441 : 스위치 패턴 매칭
